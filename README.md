# HEOR Modeling Project (Hypothetical Data)

This repository contains a set of Health Economics and Outcomes Research (HEOR) models built using hypothetical data to demonstrate key economic evaluation techniques in healthcare.

## Overview
The project focuses on analyzing cost-effectiveness, budget impact, and patient outcomes using different modeling approaches commonly used in HEOR.

## Models Included
- **Budget Impact Model** – Estimates the financial impact of introducing a new intervention.
- **Decision Tree Model** – Evaluates short-term clinical and economic outcomes.
- **Markov Model** – Models long-term disease progression and treatment pathways.
- **Partitioned Survival Model** – Used for survival analysis in healthcare studies.
- **Patient-Level Simulation Model** – Simulates individual patient variability and outcomes.

## Objective
To understand and apply various HEOR methodologies for data-driven decision-making in healthcare.

## Note
All analyses are based on hypothetical data and are intended for educational and demonstration purposes only.
